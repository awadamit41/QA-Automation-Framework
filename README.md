# QA Automation Framework

A Selenium-based QA Automation Framework developed using Java, TestNG, Maven, and Microsoft Edge Driver for automated web application testing.

## Features

- Automated Login Testing
- Selenium WebDriver Integration
- TestNG Execution & Reporting
- Maven Dependency Management
- Reusable Test Components
- Edge Browser Automation
- Git & GitHub Version Control

## Technology Stack

| Technology | Purpose |
|------------|---------|
| Java 25 | Programming Language |
| Selenium WebDriver | Browser Automation |
| TestNG | Test Execution & Reporting |
| Maven | Build & Dependency Management |
| EdgeDriver | Browser Driver |
| Git | Version Control |
| GitHub | Source Code Management |
| Eclipse IDE | Development Environment |
