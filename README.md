# QA Automation Framework

A Selenium-based test automation framework built using Java, TestNG, Maven, and WebDriver.

## Technologies Used
- Java 25
- Selenium WebDriver
- TestNG
- Maven
- Git & GitHub
- Microsoft Edge Driver

## Features
- Automated Login Testing
- Page Object Model (POM)
- Data-Driven Testing
- TestNG Reports
- Screenshot Capture on Failure
- Reusable Test Components
